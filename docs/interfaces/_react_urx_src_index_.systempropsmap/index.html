Optional ssrProps#

â€¢ ssrProps? : K[]

Defined in react-urx/src/index.ts:96

Specifies the streams which should get their values from props during server side rendering. By default, property values are transferred to their respective streams in an useEffect hook. The SSR ones will be set in the function component body. Be careful using SSR props, and use them only for "static" props. If setting an SSR prop re-renders a child component, React will throw an exception.
